# Bento Grid

`BentoGrid` is a layout system that allows you to create dense, visually interesting feature grids. It's heavily inspired by modern landing page designs.

## How To Use

<Steps>
  <Step>
    ### Install

    Run the following command to add this component to your project.

    <PackageInstall name={'bento-grid'} />
  </Step>
  <Step>
    ### Import

    Import the component into your project.

    <DynamicCodeBlock lang={'tsx'} code={`import { BentoGrid, BentoCard } from '@/components/ui/hatch/bento-grid';`}/>
  </Step>
</Steps>
