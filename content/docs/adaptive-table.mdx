# Adaptive Table

The `AdaptiveTable` solves the biggest problem with data tables: **mobile responsiveness**. Instead of forcing horizontal scrolling, it cleverly collapses into actionable cards on smaller screens.

## How To Use

<Steps>
  <Step>
    ### Install

    Run the following command to add this component to your project.

    <PackageInstall name={'adaptive-table'} />
  </Step>
  <Step>
    ### Import

    <DynamicCodeBlock lang={'tsx'} code={`import { AdaptiveTable } from '@/components/ui/hatch/adaptive-table';`}/>
  </Step>
</Steps>
