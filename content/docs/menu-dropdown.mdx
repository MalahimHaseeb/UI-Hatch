# Dropdown Menu

`DropdownMenu` is a versatile dropdown component that allows you to pass **any trigger element** (like `div`, `Box`, `Button`, etc.) and fully customize the dropdown menu.

## How To Use

<Steps>
  <Step>
    ### Install

    Run the following command in terminal to add these components in your project.

    <PackageInstall name={'smart-dropdown'} />
  </Step>
  <Step>
    ### Import

    You can now import this component anywhere you need to use it.

    <DynamicCodeBlock lang={'tsx'} code={`import { DropdownMenu } from '@/components/ui/hatch/smart-dropdown';`}/>
  </Step>
</Steps>

## Props

<TypeTable
  type={{
    triggerElement: {
      description: 'React element that will trigger the dropdown when clicked',
      type: 'React.ReactElement',
      required: true
    },
    menuItems: {
      description: 'Array of custom menu items to display in the dropdown',
      type: 'MenuItem[]',
      default: '[]'
    },
    includeDeviceUpload: {
      description: 'Whether to include the default "Upload from device" option',
      type: 'boolean',
      default: 'true'
    },
    fileInputId: {
      description: 'ID of the hidden file input element (used with includeDeviceUpload)',
      type: 'string',
      default: '"fileInput"'
    },
    className: {
      description: 'Additional className for the dropdown content',
      type: 'string'
    },
    align: {
      description: 'Alignment of the dropdown relative to the trigger',
      type: '"start" | "center" | "end"',
      default: '"end"'
    }
  }}
/>
