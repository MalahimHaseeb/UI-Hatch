# Spotlight Card

`SpotlightCard` is a modern UI element that adds a dynamic glow effect following the user's mouse. Perfect for feature sections and landing pages.

## How To Use

<Steps>
  <Step>
    ### Install

    Run the following command to add this component to your project.

    <PackageInstall name={'spotlight-card'} />
  </Step>
  <Step>
    ### Import

    Import the component into your project.

    <DynamicCodeBlock lang={'tsx'} code={`import { SpotlightCard } from '@/components/ui/hatch/spotlight-card';`}/>
  </Step>
</Steps>
