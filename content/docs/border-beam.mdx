# Border Beam

`BorderBeam` adds a sleek, animated border effect to your components. It uses hardware-accelerated CSS animations for a smooth experience.

## How To Use

<Steps>
  <Step>
    ### Install

    Run the following command to add this component to your project.

    <PackageInstall name={'border-beam'} />
  </Step>
  <Step>
    ### Import

    Import the component into your project.

    <DynamicCodeBlock lang={'tsx'} code={`import { BorderBeam } from '@/components/ui/hatch/border-beam';`}/>
  </Step>
</Steps>

## Props

<TypeTable
  type={{
    size: {
      description: 'The size/length of the beam',
      type: 'number',
      default: '200'
    },
    duration: {
      description: 'The duration of one full cycle in seconds',
      type: 'number',
      default: '15'
    },
    borderWidth: {
      description: 'The thickness of the border',
      type: 'number',
      default: '1.5'
    },
    colorFrom: {
      description: 'Starting color of the gradient beam',
      type: 'string',
      default: '#ffaa40'
    },
    colorTo: {
      description: 'Ending color of the gradient beam',
      type: 'string',
      default: '#9c40ff'
    },
    delay: {
      description: 'Animation delay in seconds',
      type: 'number',
      default: '0'
    }
  }}
/>
